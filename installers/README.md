# Installers

This folder keeps the AutoHotkey v2 installer needed to bootstrap this repo on a fresh Windows machine.

## AutoHotkey v2

- Version: 2.0.26
- File: `AutoHotkey_2.0.26_setup.exe`
- Source: https://github.com/AutoHotkey/AutoHotkey/releases/tag/v2.0.26
- Official download: https://github.com/AutoHotkey/AutoHotkey/releases/download/v2.0.26/AutoHotkey_2.0.26_setup.exe
- SHA256: `2BF1B89B1047136490FC321D2FDC988B42DD86F693EEA7872746AC6ADF722BC3`

Run the installer directly, or install silently from PowerShell:

```powershell
.\installers\AutoHotkey_2.0.26_setup.exe /silent
```
